1. Resumo sobre HTML, CSS, JS e Internet

Internet

A internet coneta os computadores do mundo inteiro e permite as pessoas acessar sites e serviços online.

HTML

HTML cria a estrutura da página, como títulos, textos, imagens e links.

Algumas tags importantes:

  • <h1> até <h6> → títulos
  • <p> → parágrafo
  • <a> → link
  • <img> → imagem
  • <ul> e <ol> → listas
  • <table> → tabela
  • <form> → formulário

Exemplo básico: 

<h1>Meu Site</h1>

<p>
  Olá mundo!
</p>

<a href="https://google.com">
  Google
</a>

CSS

CSS é usado para estilizar o site, mudando cores, tamanhos e layouts.

Exemplo básico: 

p {
  color: blue;
  font-size: 20px;
}

JavaScript

JavaScript adiciona interatividade e funcionalidades aos sites.

Exemplo básico: 

alert("Olá mundo!");

HTML, CSS e JavaScript juntos formam a base do desenvolvimento web.
